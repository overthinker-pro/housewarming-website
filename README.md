# Housewarming Website - Modern Blue Theme

## 🎉 Website Overview
A beautifully designed, modern blue-themed housewarming invitation website with professional center-aligned layout and seamless element integration.

## ✨ Key Features

### 🎨 Design & Aesthetics
- **Modern Blue Theme**: Sophisticated blue-gray color palette
- **Elegant Typography**: Sekuya for main headings, Plus Jakarta Sans for body text, Dancing Script for bible verses
- **Blue Color Scheme**: Light blue-gray (#f4f7fa), soft blue-gray (#e8eef3), deep navy (#1d3557), slate blue (#2f3e46), medium blue (#457b9d), light cyan (#a8dadc)
- **Professional Layout**: All elements perfectly center-aligned
- **Seamless Integration**: Family name and bible verse blend naturally without bordered containers
- **Responsive Design**: Perfect on all devices (mobile, tablet, desktop)
- **Smooth Animations**: Fade-in effects, hover transitions, and floating elements

### 🏠 Hero Section
- Full-screen hero with background house image
- Elegant invitation text with family name highlight
- **Integrated Event Details**: Date, time, location, and dress code displayed in organized grid
- **Bible Verse**: Psalm 127:1 displayed in beautiful Dancing Script font (no bordered container)
- Floating home icon decoration
- **Center-Aligned Layout**: Every element professionally centered
- **Natural Integration**: Family name and bible verse blend seamlessly with the design

### 🗺️ Location Integration
- Embedded Google Maps
- Direct link to open in Google Maps app
- Clean, bordered map container

### 💬 Contact & RSVP
- WhatsApp integration with pre-filled messages
- Direct call buttons
- Multiple contact options with hover effects

### ⚡ Performance & UX
- Fast loading with optimized assets
- Smooth scrolling and animations
- Scroll progress indicator
- Touch-friendly mobile interactions
- Accessibility features (keyboard navigation, focus states)
- Print-friendly styles

## 🛠️ Technical Stack
- **HTML5**: Semantic structure with proper meta tags
- **CSS3**: Modern features including CSS Grid, Flexbox, Custom Properties, and Animations
- **JavaScript ES6+**: Vanilla JS with Intersection Observer, smooth scrolling, and performance optimizations
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Sekuya, Plus Jakarta Sans, Dancing Script

## 📱 Responsive Breakpoints
- **Desktop**: 1200px+ (full layout with grid)
- **Tablet**: 768px - 1199px (optimized grid)
- **Mobile**: < 768px (single column, touch-friendly)

## 🎨 Color Palette
- **Primary**: #f4f7fa (Light Blue-Gray)
- **Secondary**: #e8eef3 (Soft Blue-Gray)
- **Text Main**: #1d3557 (Deep Navy)

## 🚀 Performance Features
- Lazy loading for images (if added)
- Optimized animations with `prefers-reduced-motion`
- Passive event listeners for better scroll performance
- Minimal JavaScript bundle
- Efficient CSS with CSS custom properties

## 🎯 User Experience
- **Single Page Application**: All content on one scrollable page
- **Progressive Enhancement**: Works without JavaScript
- **Fast Interactions**: Instant feedback on all clickable elements
- **Intuitive Navigation**: Clear visual hierarchy and call-to-actions

## 📞 Contact Integration
- **WhatsApp**: Pre-filled RSVP messages
- **Phone**: Direct call links
- **Maps**: Integrated location services

## 🔧 Customization
The website is easily customizable:
- Colors: Modify CSS custom properties in `:root`
- Content: Update text in HTML sections
- Images: Replace `images/house.png` with your background
- Fonts: Already using your specified fonts
- Contact: Update phone numbers and addresses
